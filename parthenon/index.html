Parthenon

Client SQL desktop moderne pour AWS Athena, inspiré de SSMS. Requêtez, explorez et exportez vos données.

⬇ Télécharger pour Windows ⬇ macOS (.dmg)

🎨 Éditeur SQL moderne

Monaco Editor (le moteur de VS Code) : coloration syntaxique et IntelliSense.

🗄️ Explorateur de bases

Parcourez bases, tables, colonnes et schémas Athena en un clic.

📊 Grille de résultats

Affichage haute performance des résultats avec mise à jour temps réel.

🔐 Authentification flexible

OKTA, profil AWS, clés d'accès ou SSO — connectez-vous comme vous voulez.

📝 Multi-onglets

Travaillez sur plusieurs requêtes en parallèle, avec vue partagée.

💾 Gestion des serveurs

Organisez vos connexions par dossiers, persistées entre les sessions.
